class LspTextEditor

package moonshine.editor.text.lsp

extends TextEditor

@:build(feathers.macros.StyleContextMacro.build())@:autoBuild(feathers.macros.StyleContextMacro.build())@:build(feathers.macros.StyleMacro.build())@:autoBuild(feathers.macros.StyleMacro.build())

Adds language code intelligence features to the TextEditor component, such as completion, signature help, hover tool tips, ctrl+click jump to definition, etc.

This component uses value objects (VOs) that are based on the Language Server Protocol (which is where the name LspTextEditor comes from). However, using this protocol is not mandatory. As long as objects are translated into LSP VOs, any other provider of code intelligence data may be used instead.
